# CAD Explorer assets (`public_html/disable/assets`)

Place the Gene Synthesizer IGES file here:

- **Filename:** `gene-synthesizer.igs`
- **Original:** Replication- Gene Synthesizer by Kong, Jacobson.igs

Obtain the file from NIH 3D Print Exchange, Thingiverse, or similar, then rename and save it in this folder.

**GLB:** You do not need to place GLB files here for the app to work. Use **Import CAD** in the viewer to load `.glb` meshes from anywhere on disk; this folder is only for bundled sample CAD (IGES above).
