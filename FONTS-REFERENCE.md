# Font choices for OSiris project

The project uses **2 font resources**:

---

## 1. **Plus Jakarta Sans** (text)
- **Type:** Geometric sans-serif
- **Weights used:** 300, 400, 500, 600, 700, 800
- **Role:** All body text, display, editorial, UI
- **Usage:** `font-display`, `font-editorial` (Tailwind), body, panels, recommendations
- **Google Fonts:** `Plus+Jakarta+Sans:wght@300;400;500;600;700;800`

---

## 2. **Material Symbols Outlined** (icons)
- **Type:** Icon font
- **Weights:** variable (100–700)
- **Role:** Icons (play, close, arrow, etc.)
- **Usage:** All pages via `material-symbols-outlined` class
- **Google Fonts:** `Material+Symbols+Outlined`
